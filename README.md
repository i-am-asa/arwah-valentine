# Will you be my Valentine, Arwah?

A small Valentine’s page for Arwah: “Will you be my valentine, Arwah?” with a run-away No button, floating hearts, and a letter with a poem.

## Run locally

Open `index.html` in a browser, or serve the folder (e.g. `python3 -m http.server 8000`) and visit `http://localhost:8000`.

## Deploy (GitHub Pages)

1. Push this folder to a GitHub repo.
2. **Settings → Pages** → Source: **Deploy from a branch** → Branch: `main`, folder **/ (root)** → Save.
3. After a couple of minutes, the site will be at:  
   **https://\<your-username\>.github.io/\<repo-name\>/**

(Replace with your actual URL and paste it here when live.)
